About Project

This is our Real Time Object Detection using Gesture Controlled Car. Our project able to detect more than 80 Objects.

  • Gesture Controlled Car is a robot which can be controlled by a human gesture. The user just need to wear a hand glove on which different sensors are placed.
  • Real time object detection as the name suggests is an art of detecting various objects at that particular time.
  • Object detection has always been a daring task. There are many approaches for detection

The sensors will record the movement of hand which will result in the motion of the car in the given direction. With the help of gesture control car, we can detect real time detection in any places. We present YOLO, a new approach to object detection. A single neural network predicts bounding boxes and then classify from the full images in one evaluation.YOLO makes more localization errors.
